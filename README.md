# Health Analyzer

A simple browser-based health analyzer that estimates wellness metrics from daily health details. The app calculates BMI, estimated maintenance calories, waist-to-height ratio, hydration progress, and habit-based insights.

## Features

- Interactive form for age, sex, height, weight, waist, activity, sleep, and water intake
- BMI category and waist-to-height risk indicator
- Estimated daily maintenance calories
- Hydration goal based on body weight
- Overall wellness score with practical habit notes
- Reset button for returning to default sample values

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

Open `index.html` in a browser.

No build step or backend server is required.

## Project Structure

- `index.html` - App layout and form fields
- `styles.css` - Responsive visual design
- `app.js` - Health calculations, scoring, and UI updates

## Note

This project is for learning and general wellness estimates only. It is not medical advice, diagnosis, or a replacement for guidance from a healthcare professional.
